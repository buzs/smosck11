## Project

Connecting people with NGO's what need for help

## Install and Run

Clone this repo in your computer and using `yarn` or `npm` to install dependencies

### Back-end

```bash
cd backend

yarn
yarn start
#or
npm install
npm start
```

### Front-end

```bash
cd frontend

yarn
yarn start
#or
npm install
npm start
```

### Mobile (Expo)

```bash
cd mobile

yarn
yarn start
#or
npm install
npm start
```
